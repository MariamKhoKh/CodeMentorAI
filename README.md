# CodeMentorAI

AI-powered coding mentor platform with a Python FastAPI backend and a React frontend.

## Features
- User authentication
- Coding problem submissions and feedback
- Automated code analysis and recommendations
- Dashboard and exercise pages

## Tech Stack
- **Backend:** Python, FastAPI, SQLAlchemy, Alembic, Docker
- **Frontend:** React, Vite, Tailwind CSS
- **Database:** (Configured via Docker Compose)

## Getting Started

### Prerequisites
- Docker & Docker Compose
- Node.js & npm (for frontend development)

### Backend
1. Navigate to `backend/`:
   ```sh
   cd backend
   ```
2. Start backend (with DB) using Docker Compose:
   ```sh
   docker-compose up
   ```

### Frontend
1. Navigate to `frontend/`:
   ```sh
   cd frontend
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Project Structure
- `backend/` — FastAPI app, models, services, migrations
- `frontend/` — React app, pages, components
